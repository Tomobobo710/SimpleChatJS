System Prompts

Configure system instructions that guide AI behavior across all conversations. Applied automatically to the first message in each conversation.

This prompt will be automatically added to the beginning of new conversations with all AI providers.

Context toggles

Appends environment info (platform, working directory, shell, date) to the system prompt. The working directory only affects freeform chats — project chats use the project root.

Anthropic (Claude) Thinking

Extended reasoning for Claude models. Choose the mode that matches your model generation.

8192

Range: 1024–32000 tokens. Must be less than max_tokens.

Google (Gemini) Thinking

Controls Gemini's internal reasoning. Choose the API mode that matches your model generation.

8192

Range: 128–32768. Use 0 to disable (Flash only), -1 for auto.

Gemini 3 Pro only supports low and high (not medium).

llama-server Thinking

Controls reasoning token allocation per request. Works with DeepSeek-R1, QwQ, Qwen3, and other reasoning-capable models.

Auto

How reasoning tokens are returned. Use "DeepSeek" to get thinking blocks in the UI.

OpenAI Reasoning

Sets reasoning effort for o-series and GPT-5 models. Has no effect on non-reasoning models.

Sent as reasoning_effort in the Chat Completions API.

SimpleTools

Built-in tools available to the AI. These run natively without external servers.

read_file Read a UTF-8 text file and return its content
write_file Create or overwrite a file with given content
edit_file Replace exact string in an existing file
shell_run Run a shell command and return output
Caps read_file output; shell_run uses 1 KB less to leave room for its truncation notice. Min 2.

Shell

Select the shell for shell_run. "Auto" detects the best available shell on save.
